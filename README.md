# Dotfiles

My private dotfiles collection, used as a reference for me and others.  
Helps me automate my environment on new systems.

**To install the dotfiles**

```
git clone https://github.com/vic3lord/dotfiles ~/.dotfiles
~/.dotfiles/bootstrap
```
